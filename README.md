JobRadar

JobRadar is an AI-assisted job application tracker that helps users manage applications, track recruiter responses, and automate parts of the job search process.

Features
Job application tracking
Status history
Dashboard & statistics
Document management (CV, Cover Letter)
Google Authentication
Gmail integration
AI-powered email analysis
AI Suggestions workflow
Automated Gmail scanning
Multi-language support (EN / DE / RU)
